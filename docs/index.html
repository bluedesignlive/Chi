Architecture

How it flows

No hardcoded values. Every component reads from the token layer. Change one source, update everywhere.

01
ChiTheme
Colors, typography, surfaces. Algorithmic M3 palette generation from any seed color with light/dark derivation.
02

Durations, easing curves, spring physics. Every animation uses shared tokens — no magic numbers.
03

Shadows and depth layers. Consistent elevation tokens for every surface in the system.

The API

It just works

Import Chi. Place components. The theme, motion, and elevation systems handle the rest.

main.qml
import Chi

ChiApplicationWindow {
    title: "My App"

    Button {
        text: "Do something"
        onClicked: console.log("done")
    }

    // Dark mode, motion, window frame — all handled
}
Deployment

Quick start

Requires Qt 6. Developed on Linux and BSD. Cross-platform in principle.

01 Clone & Install

git clone https://github.com/bluedesignlive/Chi.git
cd chi
./install.sh

02 Use in QML

import Chi

ChiApplicationWindow {
    Button { text: "Hello" }
}
