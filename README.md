# Chi



Chi is built on a simple belief: open-source desktops deserve the same care that went into the software underneath them.

Unix systems are powerful, stable, and free. They should also be beautiful, personal, and alive. Not because beauty is superficial — but because the way software looks and moves is how it communicates with the person using it.

### What Chi Is
**A modern UI library for Unix-based systems.**
Built to make apps feel timeless, elegant, expressive, and deeply human.

A clean, modern component system that brings together:

- **Performance** — transform-only animations, gated GPU layers, no unnecessary redraws
- **Reliability** — modern Qt6, zero legacy dependencies, production-ready components
- **Clarity** — declare what you want, Chi handles the rest
- **Customizability** — one theme singleton controls colors, typography, motion, and density globally
- **Good design principles** — every component follows Dieter Rams' ten principles of good design
- **A modern visual language** — Material Design 3 tokens, three density tiers, dark and light themes

By providing this, Chi aims to help developers build applications that feel
intuitive, refined, and timeless — and in doing so, bring fresh energy to
the Unix software landscape.
